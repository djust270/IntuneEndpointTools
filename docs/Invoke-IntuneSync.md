# Invoke-IntuneSync

## SYNOPSIS
Force a device check-in with Intune

## SYNTAX

```
Invoke-IntuneSync
```

## DESCRIPTION
Force device check-in by running the associated scheduled tasks created by Intune and restarting the Intune Management Extension.

## EXAMPLES

### Example 1
```powershell
PS C:\> Invoke-IntuneSync
```
